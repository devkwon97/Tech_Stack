## 클래스와 객체

### 클래스(Class)
<hr>
클래스란 객체를 정의해 논 것  

객체의 설계도 또는 틀이라고 정의 가능  
객체를 생성하는데 사용되며, 객체는 클래스에 정의된 대로 생성 됨  
ex)마치 붕어빵을 찍는 붕어빵 틀

### 객체(Object)
객체란 실제로 존재하는 것  
눈으로 볼 수 있는 형태 개념이나 논리가 같은 형태 없는 것도 객체로 간주

<img src="../resources/instant.jpg" width="100%" height="20%">
객체와 인스턴스는 같은 의미지만  
객체는 모든 인스턴스를 대표하는 포괄적 의미 (실체화된 인스턴스는 메모리에 할당)
